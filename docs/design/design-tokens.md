# Mavora Labs Design Tokens

## Color

| Token | Value | Usage |
| --- | --- | --- |
| Background | `#fbfaf7` | Page background |
| Surface | `#ffffff` | Cards and high-contrast content blocks |
| Text | `#111827` | Primary text |
| Muted text | `rgba(17, 24, 39, 0.68)` | Secondary copy |

| Token | Value | Usage |
| --- | --- | --- |
| xs | 0.5rem | Tight gaps |
| sm | 1rem | Card content gaps |

| lg | 2.5rem | Card grids and blocks |
| xl | 5rem | Section vertical padding |

## Shape And Effects

- Border radius: 8px cards, full radius pill CTAs
- Border color: `#d9ded7`
- Shadow style: soft shadows only for service/package cards
- Icon style: Lucide line icons
- Motion style: short hover transitions, no required motion

## Density

- Marketing/content density: clear, scannable sections with concise copy
- App/table density: not applicable
- Mobile density notes: stack cards, keep CTAs full-width-friendly, avoid overlapping sticky UI
