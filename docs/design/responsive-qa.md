# Mavora Labs Responsive QA

## Viewports

Check at minimum:

- 360px mobile
- 390px mobile
- 768px tablet
- 1024px laptop
- 1440px desktop

## Layout Checks

- Header and navigation work.
- Text does not overlap controls or neighboring content.
- Buttons fit their labels.
- Hero image preserves useful framing.
- Sticky/fixed header does not block anchor content excessively.

## Interaction Checks

- CTA reaches the contact section for now.
- Final WhatsApp link works after replacement.
- Keyboard focus is visible.
- Touch targets are large enough.
