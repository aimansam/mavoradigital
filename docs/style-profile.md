# Landing Page Profile

## Must Define

- Primary offer: freelance website design and development for service businesses.
- Primary audience: freelancers, local businesses, and growing teams.
- Conversion action: start a project, later routed to WhatsApp.
- Traffic source: direct referrals, social profiles, and portfolio links.
- Form destination: none for v1; WhatsApp CTA after final setup.
- Success metric: qualified conversations and project inquiries.

## QA Focus

- Mobile layout
- CTA link replacement
- Analytics event when analytics is added
- SEO and social preview
- Page speed
