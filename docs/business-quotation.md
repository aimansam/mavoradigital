## Project Summary

Mavora Labs collects website project requests through a static form that opens WhatsApp with the selected project details. The request captures the client's business type, page needs, domain status, content readiness, feature requirements, timeline, and budget approach.

## Request Inputs

- Business type and target customers
- Pages or sections needed
- Content and image readiness
- Domain ownership status
- Required features or integrations
- Preferred timeline
- Budget range or project priority

## WhatsApp Request Form

The public homepage `/#project-request` section includes a static request form. The client can choose:

- Website type
- Business type
- Timeline
- Budget approach
- Domain ownership status
- Pages or sections needed
- Features or setup needed
- Content readiness
- Reference website
- Main website goal
- Extra project notes

Submitting the form opens WhatsApp using `PUBLIC_WHATSAPP_NUMBER`.

The form includes a short privacy message with a `View privacy note` link to `/privacy/`: client details are only used to respond to the website request.
